解决nuxt.js中使用element-ui默认语言为英文

2020-08-23

都知道更改 Element-UI 的语言设置，把 import locale from 'element-ui/lib/locale/lang/en' 改为 import locale from 'element-ui/lib/locale/lang/zh-CN' 即可。但是在哪个文件改呢，查看 nuxt.config.js 可以看到使用 element-ui 的地方在 plugins 里面，所以要改的文件就是 plugins/element-ui.js 。
